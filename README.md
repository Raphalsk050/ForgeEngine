# ForgeEngine


This engine project is being built completely from scratch, integrating some of the most powerful open-source libraries available. 
For the graphical interface, I'm using ***Dear ImGui*** in conjunction with ***Walnut***, for physics I'm utilizing the ***Bullet3 physics SDK***, and for graphics, we're employing ***Vulkan***.

This Engine project is currently made from scratch 

## Requirements
- [Visual Studio 2022](https://visualstudio.com) (not strictly required, however included setup scripts only support this)
- [Vulkan SDK](https://vulkan.lunarg.com/sdk/home#windows) (preferably a recent version)
